## 💡 Troubleshooting Common Issues

If the software fails to open or your connections drop, try these steps first.

1. Restart your computer. This clears minor conflicts.
2. Check your internet connection status. 
3. Run the installer again to perform a repair.
4. Look for the latest version on the download page. Older versions might lack updates for new Windows security patches.

If a specific session will not connect, verify the address and port number. Typos are the most common cause of failed connections. Ensure the machine on the other side of the connection is turned on and reachable.
